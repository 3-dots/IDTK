IDTK

In-Design Tool-Kit

What?!


(What is this?!) IDTK (In-Design Tool-Kit) is an extension for Adobe InDesign CS.

As the name suggests, it is intended to be a "Tool-Kit"! But right now it does just one thing! (something really cool tho!) We just released the beta version of one of the tools (modules) of the toolkit which we call “Robot”!
We call it Robot because it is like a Robot! A designer robot! A D-Bot! It's a worker robot who designs graphical elements for your pages!

You just need to tell him what you want and it will generate customized, beautiful patterns and motifs tailored for your document! But wait! Anybody can do that! (OK! not anybody! But a good graphic designer can do) But it's not all it can do for you! In fact it works on every single page of your document and makes a unique version of the selected patterns for each and every single page of your documents (you can select which pages too), something normal "human" graphic designers just can't (or if they do, they will need days and weeks and… even months if you have hundreds of hundreds of pages!), so each page will be different than the others! Because otherwise that'd be boring! Wouldn't it?! Whatever! Just let the Robot help you!

About the other tools, We already have lots of ideas and we are impatient to start working on them but it all depends on how the community receives our work! We worked really hard and built this tool up from the ground with pure love and we need your support to continue doing so! ❤

How?!


- How it works? With about 5000 lines of code written in 5-6 different programming languages, it uses mathematical algorithms to generate (unique) patterns and motifs in an real-time (on-air) fashion for your documents!

- How to install it?! Just download the extension file and double-click on it and let the Adobe Extension Manager do the rest of the job! It will automatically install the product and will also inform you later-on directly in it's UI if an update available is available! If you want to see a video-tutorial about how to install the plugin, click here!

- How to run it?! After installation, all you need to do is to open InDesign and you'll see you got a new menu item up there, titled " * IDTK "! you can access all the IDTK functionalities through that menu.

- How to use it?! Usage of the plug-in is pretty easy but you can check out these videos as well, Please just remember to save and make a backup of your project before running any extension (including ours, specially during the beta release!).
